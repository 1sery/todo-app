## 在线演示

[点击这里访问在线演示](https://lsery.github.io/todo-app/)

## 本地运行

1. 克隆仓库
   ```bash
   git clone https://github.com/lsery/todo-app.git
   cd todo-app
   ```
2. 安装依赖
   ```bash
   npm install
   ```
3. 启动开发服务器
   ```bash
   npm run dev
   ```
4. 打开浏览器访问 `http://localhost:5173`
